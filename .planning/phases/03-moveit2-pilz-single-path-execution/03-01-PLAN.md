<objective>
Wire ROS2 dependency installation into the devcontainer first-start flow and integrate
`MoveItPy` into `URMovementController`.

`moveit_py` is already declared as `<exec_depend>moveit_py</exec_depend>` in `package.xml`.
Rather than hardcoding `ros-jazzy-moveit` in the Dockerfile, a `docker/startup.sh` script
runs via `postCreateCommand` and calls `rosdep install --from-paths src --ignore-src -r -y`
against the live mounted workspace — so `package.xml` is the single source of truth for all
ROS2 dependencies and `moveit_py` is installed automatically on first devcontainer start.

Then wire `MoveItPy` into `URMovementController`: add the instance fields in `__init__`,
declare all three parameters in `__init__` (including `moveit_connection_timeout`), probe
`move_group` readiness in `on_configure` via `wait_for_service` then call `MoveItPy()` directly
inside a `try/except` (returning FAILURE on timeout or exception), and tear it down in `on_cleanup`.

Purpose: All subsequent Phase 3 plans depend on `moveit_py` being importable and on
`self._moveit` / `self._planner_service` fields existing in the controller.

Output: `docker/startup.sh`, updated `devcontainer.json` postCreateCommand, and updated
`ur_movement_controller.py` with full MoveItPy lifecycle.
</objective>

MoveItPy init pattern in on_configure:

  # Step 1: probe move_group service to detect bad launch before blocking on MoveItPy init
  from moveit_msgs.srv import GetPlanningScene
  client = self.create_client(GetPlanningScene, '/move_group/get_planning_scene')
  if not client.wait_for_service(timeout_sec=timeout):
      self.get_logger().error(
          f'move_group not available after {timeout}s — is move_group running?'
      )
      self.destroy_client(client)
      return TransitionCallbackReturn.FAILURE
  self.destroy_client(client)
  # Step 2: init MoveItPy directly — service check already confirmed move_group is up
  try:
      self._moveit = MoveItPy(node_name='moveit_py_node')
      planning_component = self._moveit.get_planning_component(moveit_group_name)
      self._planner_service = PilzPlannerService(self._moveit, planning_component)
  except Exception as e:
      self.get_logger().error(f'MoveItPy initialisation failed: {e}')
      return TransitionCallbackReturn.FAILURE
  self.get_logger().info(f'MoveItPy connected; planning group: {moveit_group_name}')
