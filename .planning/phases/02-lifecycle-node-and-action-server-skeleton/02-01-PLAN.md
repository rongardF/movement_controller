  **Header:** Copy the exact BSD-3-Clause header from `src/movement_controller/movement_controller/__init__.py` lines 1–28. Add a module docstring: `"""URMovementController — ROS2 LifecycleNode for trajectory execution."""`

  **Imports (in this order):**
  - stdlib: `import threading`
  - rclpy: `import rclpy`, `from lifecycle_msgs.msg import State`, `from rcl_interfaces.msg import ParameterDescriptor`, `from rclpy.action import ActionServer`, `from rclpy.lifecycle import LifecycleNode`, `from rclpy.lifecycle.node import LifecycleState, TransitionCallbackReturn`

  **Class definition:** `class URMovementController(LifecycleNode):`

  **`__init__`:** Signature `def __init__(self, node_name: str = 'ur_movement_controller') -> None:`. Calls `super().__init__(node_name)`. Initialises three instance variables:
  - `self._action_server: ActionServer | None = None`
  - `self._is_executing: bool = False`
  - `self._executing_lock = threading.Lock()`

  **`on_configure`:** Signature `def on_configure(self, state: LifecycleState) -> TransitionCallbackReturn:`. Logs `f'Configuring from state: {state.label}'` at INFO level. Declares two parameters using `self.declare_parameter(name, default, ParameterDescriptor(description=...))`:
  - `'action_server_name'` with default `'movement_controller/execute_trajectory'` and description `'ROS2 action server name for ExecuteTrajectory action'`
  - `'moveit_group_name'` with default `'ur_manipulator'` and description `'MoveIt2 planning group name (used from Phase 3 onward)'`
  Returns `TransitionCallbackReturn.SUCCESS`.

  **`on_activate`:** Logs `f'Activating from state: {state.label}'` at INFO. Returns `TransitionCallbackReturn.SUCCESS`.

  **`on_deactivate`:** Logs `f'Deactivating from state: {state.label}'` at INFO. Returns `TransitionCallbackReturn.SUCCESS`.

  **`on_cleanup`:** Logs `f'Cleaning up from state: {state.label}'` at INFO. If `self._action_server is not None`, calls `self._action_server.destroy()` then sets `self._action_server = None`. Returns `TransitionCallbackReturn.SUCCESS`.

  **`main` function:** `def main(args=None) -> None`. Calls `rclpy.init(args=args)`, creates `node = URMovementController()`, calls `rclpy.spin(node)`, then `node.destroy_node()`, then `rclpy.shutdown()`. Wrap the spin in a `try/finally` so `destroy_node` and `shutdown` always run.

  **CRITICAL:** Do NOT return `None`, `True`, or any integer from lifecycle callbacks — only `TransitionCallbackReturn.SUCCESS` or `TransitionCallbackReturn.FAILURE`. Do NOT create an ActionServer in `__init__` — that is Plan 02-02.
