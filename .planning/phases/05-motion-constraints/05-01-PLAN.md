## Objective

Create the `ConstraintConfigDTO` Pydantic model, declare all 15 constraint parameters in
`URMovementController.__init__`, read + validate them on `on_configure`, and add the
`set_constraints()` stub to `PilzPlannerService`. This is the foundation every downstream plan
(02–05) builds on.

Purpose: All persistent motion constraint configuration flows through a single validated DTO
(D-01, D-09). Parameters are declared once in `__init__` and read fresh on every `on_configure`
(D-10).

Output: New `constraint_config_dto.py`, updated `models/__init__.py`, updated
`ur_movement_controller.py` (params + on_configure), updated `pilz_planner_service.py`
(`set_constraints` stub + `_constraint_config` attribute).

    Create a new file with BSD-3-Clause license header and module docstring
    "ConstraintConfigDTO — Pydantic v2 DTO for persistent motion constraints."

    Import: `from __future__ import annotations`, `import math`, `from pydantic import BaseModel,
    ConfigDict, Field, model_validator`.

    Define `class ConstraintConfigDTO(BaseModel):` with `model_config = ConfigDict(frozen=True)`.

    Fields (all require `Field(description=...)`):

    Workspace bounding box (float, use D-12 sentinels -1e9 / +1e9 as defaults):
    - `x_min: float = Field(default=-1e9, description='...')`
    - `x_max: float = Field(default=1e9, description='...')`
    - `y_min: float = Field(default=-1e9, description='...')`
    - `y_max: float = Field(default=1e9, description='...')`
    - `z_min: float = Field(default=-1e9, description='...')`
    - `z_max: float = Field(default=1e9, description='...')`

    Joint constraints (use default_factory=list):
    - `joint_names: list[str] = Field(default_factory=list, description='Joint names for position constraints; empty = no joint constraints.')`
    - `joint_lower_limits: list[float] = Field(default_factory=list, description='Lower position limits in radians, matching joint_names order.')`
    - `joint_upper_limits: list[float] = Field(default_factory=list, description='Upper position limits in radians, matching joint_names order.')`
    - `joint_max_velocities: list[float] = Field(default_factory=list, description='Max joint velocities in rad/s, matching joint_names order. Stored for documentation; PILZ has no per-joint velocity API (D-13).')`

    Orientation constraint (default = 2*math.pi = unconstrained per D-04):
    - `orientation_tolerance_x: float = Field(default=2 * math.pi, description='...')`
    - `orientation_tolerance_y: float = Field(default=2 * math.pi, description='...')`
    - `orientation_tolerance_z: float = Field(default=2 * math.pi, description='...')`

    Speed limits (default = 0.0 = unconstrained per D-04):
    - `max_cartesian_speed: float = Field(default=0.0, description='Node-level max cartesian speed cap. 0.0 = unconstrained. Goals with path.cartesian_speed exceeding this are rejected at _goal_callback.')`
    - `max_acceleration: float = Field(default=0.0, description='Node-level max acceleration cap. 0.0 = unconstrained.')`

    Add two `@model_validator(mode='after')` methods with distinct names:

    `_validate_workspace_bounds`: Raise `ValueError` if x_min > x_max, y_min > y_max, or
    z_min > z_max. Include both values in the message.

    `_validate_joint_arrays`: Compute lengths of joint_names, joint_lower_limits, joint_upper_limits.
    If any is non-empty and they differ in length, raise `ValueError` with all three lengths.
    Also validate joint_max_velocities: if non-empty, must match len(joint_names).

    Add three `@property` methods (these are plain Python properties, not Pydantic validators):

    `workspace_enabled -> bool`: Returns `True` when the workspace is NOT at full sentinel range.
    Disabled condition (return False) = ALL three axes have range >= 2e9:
    `not (self.x_max - self.x_min >= 2e9 and self.y_max - self.y_min >= 2e9 and self.z_max - self.z_min >= 2e9)`.

    `joint_constraints_enabled -> bool`: Returns `len(self.joint_names) > 0`.

    `orientation_constraint_enabled -> bool`: Returns `True` if any tolerance < 2*math.pi
    (i.e., `self.orientation_tolerance_x < 2 * math.pi or self.orientation_tolerance_y < 2 * math.pi or self.orientation_tolerance_z < 2 * math.pi`).
