## Verification

```bash
# Verify directory structure:
find movement_controller/ -name "*.py" | sort

# Verify no ur_movement_controller.py was created:
ls movement_controller/ur_movement_controller.py 2>&1 | grep "No such file"

# Verify BSD headers on all __init__.py files:
grep -r "Copyright" movement_controller/

# Verify no stray content (no import, no __all__, no class):
grep -r "^import\|^from\|^class\|^def\|__all__" movement_controller/ || echo "Clean — no stray content"

# After build, test importability:
source /opt/ros/jazzy/setup.bash
colcon build --symlink-install --packages-select movement_controller
source install/setup.bash
python3 -c "import movement_controller; from movement_controller import models, enums, utils, services; print('All sub-packages importable')"
```
